FUNCTION ValidateToken:

  AMBIL header Authorization

  JIKA kosong:
    RETURN response:
      valid = false
      message = "Token tidak ditemukan" (401)

  HAPUS prefix "Bearer " dari token

  JIKA format salah:
    RETURN response:
      valid = false
      message = "Format token salah" (401)

  PANGGIL usecase.ValidateToken(token)

  JIKA error:
    RETURN response INTERNAL_ERROR (500)

  JIKA token tidak valid:
    RETURN response tidak valid (401)

  RETURN response valid (200)
